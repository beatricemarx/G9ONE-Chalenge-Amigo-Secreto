<h2>Descrição do Projeto </h2>
      <p> Projeto criado durante a participação do Programa One - Oracle Next Education - Turma 9. O objetivo é desenvolver as habilidades em lógica de programação, com a criação de um jogo em Javascript aonde o usuário pode inserir nomes (criando uma array)  e depois realizar um sorteio aleatório de um destes nomes. </p>

<h2>Funcionalidades do Projeto </h2>
      <p>Trata-se de um jogo aonde o usuário pode inserir nomes de seus amigos e depois realizar um sorteio aleatório de um destes nomes. </p>

<h3>Inserir os nomes </h3>
      <p>Passo 1: Para inserir os nomes dos seus amigos basta escrever no campo aonde está escrito "Digite um nome" e clicar no botão "Adicionar". </p>

      <p>Passo 2: O nome digitado aparecerá abaixo do botão "Digite um nome", siginificando que ele entrou para a lista do sorteio.</p>
